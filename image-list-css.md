================================================================================
OBSERVAÇÃO SOBRE RETINA
================================================================================

Retina = Telas de alta densidade de pixels (2x)
- Tela normal: 1 pixel CSS = 1 pixel físico
- Tela Retina: 1 pixel CSS = 2x2 pixels físicos (4 pixels físicos)

Para telas Retina, você pode criar versões @2x (2x maior) para máxima nitidez.
Se não criar versões @2x, use o tamanho "Recomendado" que já funciona bem.

================================================================================
DICAS DE OTIMIZAÇÃO
================================================================================

1. FORMATO:
   - Use WebP para melhor compressão (70-80% menor que PNG)
   - Mantenha PNG apenas se precisar de transparência
   - JPG para fotos sem transparência

2. QUALIDADE:
   - WebP: qualidade 80-85
   - JPG: qualidade 80-85
   - PNG: use compressão otimizada

3. FERRAMENTAS:
   - ImageMagick: convert input.png -resize 1920x1080 output.webp
   - Squoosh.app (online): https://squoosh.app
   - Sharp (Node.js): npm install sharp

4. ORDEM DE PRIORIDADE:
   - Primeiro: Hero images (maior impacto)
   - Segundo: Imagens de conteúdo
   - Terceiro: Cards e equipe
   - Por último: Testemunhos e outras
